Programming Terms

Learn about the foundational terms of web programming.

Key Terms

  • DOM: The Document Object Model allows JavaScript to dynamically interact with and manipulate web content.
  • API: Application Programming Interfaces let applications communicate and exchange data.
  • CSS Grid: A layout system for designing web pages with rows and columns.
  • Favicon: The small icon that appears in the browser tab for a website.
  • Responsive Design: A design approach ensuring websites work well on all screen sizes.
