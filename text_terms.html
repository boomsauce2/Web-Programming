Web Programming Terms

Key Terms

  • HTML: The backbone of web pages, providing structure with elements like headers, paragraphs, and links.
  • CSS: Styles web pages, controlling color, layout, and fonts to improve user experience.
  • JavaScript: Adds interactivity to web pages, such as animations, form validation, and dynamic content updates.
  • Responsive Design: Designing websites that work well on screens of all sizes, from desktops to phones.
  • GitHub Pages: A platform to host and share websites directly from Git repositories.
