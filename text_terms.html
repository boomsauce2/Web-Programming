Web Programming Terms

  • HTML: The backbone of web pages, providing structure with elements like headers, paragraphs, and links.
  • CSS: Controls the style and layout of web pages, including colors, fonts, and responsive design.
  • JavaScript: A programming language that adds interactivity, like animations or dynamic content.
  • Responsive Design: Ensures websites work seamlessly on devices of all sizes, from desktops to smartphones.
  • GitHub Pages: A free service to host and share static websites directly from your Git repository.
